#  도키도키 미소년 연애 시뮬레이션

> 현실에선 백수지만 — 이 세계에서는 미소년들에게 사랑받는 메인 히로인?!
> 당신의 선택, 당신의 말, 당신의 호감도… 모든 것이 엔딩을 바꾼다.

연애 시뮬레이션과 비주얼 노벨의 조합, 감성 UI + BGM + 호감도 시스템까지 갖춘 Web 기반 인터랙티브 게임 프로젝트입니다.
**별도 설치 없이 브라우저만 있으면 즉시 플레이 가능!**
(하지만 로컬 서버 실행 시 오디오·비디오 안정성이 더 좋아집니다.)

## 핵심 메커니즘 — **게임이 돌아가는 원리**

### 1. **플레이어 이름 입력 → 변수로 저장 → 스토리 내 자동 치환**

```js
"{playerName}.. 예쁜 이름이네..."
```

대사 안에 `{playerName}`만 쓰면 즉시 반영됨.
**→ 몰입감 x 200% 증가**

---

### 2. **호감도(Affection) 시스템**

선택지에 따라 캐릭터별 점수가 누적됨.

```js
affection: { Noah: +2}
```

엔딩 분기 예:

| 호감도   | 엔딩         |
| ----- | ---------- |
| 10 이상 | ❣ 고백 루트    |
| 5~9   | ✨ 친구 이상 썸  |
| 4 이하  | 💔 BAD END |

> → 이 시스템만 잘 확장하면 **멀티루트/히든엔딩/폭력적 NTR/극락해피엔드** 전부 구현 가능.

---

### 3. **씬 구조**

story[] 내부 객체 하나가 *하나의 장면*을 의미.

```js
{
  id: "music_class",
  type: "choice",
  question: "기타 소리가 울린다. 당신의 반응은?",
  choices: [
    { text: "조용히 근처에 앉아 듣는다", next: "approach", affection:{ Noah:+1 }},
    { text: "같이 연주하고 싶다고 말한다", next: "playTogether", affection:{ Noah:+3 }},
    { text: "그냥 교실을 나간다", next: "badMood", affection:{ Noah:-2 }},
  ]
}
```

지원하는 type

| 타입          | 설명                   |
| ----------- | -------------------- |
| `dialogue`  | 일반 텍스트 출력            |
| `narration` | 장면 설명, 분위기 연출        |
| `choice`    | 선택지 생성 → next ID로 연결 |
| `video`     | mv.mp4 재생            |
| `gameover`  | 사망·파국 루트             |
| `ending`    | 스페셜 엔딩               |
